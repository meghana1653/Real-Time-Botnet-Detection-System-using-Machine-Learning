
#🛡️ Real-Time Botnet Detection using Machine Learning
📌 Overview

This project detects Botnet attacks in real-time using Machine Learning.
The system captures network traffic, extracts features, trains a model, and identifies malicious activity automatically.

🎯 Features
Real-time network traffic monitoring
Machine learning-based detection
Automatic botnet detection
Live dashboard visualization
Flow-based traffic analysis

🧠 Technologies Used
Python
Scikit-learn
Scapy
Pandas
Streamlit
FastAPI

⚙️ Workflow
Capture Network Traffic
Feature Extraction
Dataset Preparation
Train Machine Learning Model
Real-Time Detection
Dashboard Monitoring

▶️ How to Run
python sniffer.py
python prepare_dataset.py
python add_features.py
python generate_flow_dataset.py
python train_model.py
python realtime_detector.py
streamlit run dashboard.py

📊 Results
Detects botnet traffic in real-time
High accuracy prediction
Live monitoring dashboard

🎯 Conclusion

This project improves network security by detecting Botnet attacks in real-time using Machine Learning.
